# Activity 1

Objectives
- Learn how to create new Spring Boot Project

What to do:
- Go to https://start.spring.io/
- Select the following configuration
    - Gradle - Groovy
    - Java 21
    - Spring Boot 3.5.0
    - Packaging - Jar
    - Dependencies:
        - Lombok
        - Spring Web
        - Spring Data MongoDB

Acceptance Criteria:
- Newly created Spring Boot Project
- Can do `./gradlew bootRun`
